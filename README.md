# よぼっちゃ 🧠

**認知症リスク診断アプリ**

> 今の自分を知って、これからの安心につなげる

かんたんチェックで、未来の自分をサポートする無料のWebアプリです。

## 📖 概要

認知症予防に関する国際的な研究知見（2024年）をもとに、「修正可能な14のリスク因子」についてセルフチェックができます。

- ✅ 登録不要・無料
- ✅ 「はい」「いいえ」を選ぶだけ（約5〜10分）
- ✅ 個別の予防アドバイスを表示
- ✅ 回答データはブラウザ内のみ処理（サーバー送信なし）

## 🚀 使い方

GitHub Pages でそのまま公開できます。

```
https://[your-username].github.io/yoboccha/
```

ローカルで確認する場合は `index.html` をブラウザで開くだけです。

## 📁 ファイル構成

```
yoboccha/
├── index.html          # メインアプリ（単一ファイルSPA）
├── assets/
│   └── yoboccha-hero.jpg  # トップイラスト
├── REQUIREMENTS.md     # 要件定義書
└── README.md
```

## ⚠️ 免責事項

本アプリは医療診断の代替ではありません。  
気になる症状がある場合は必ず医療機関にご相談ください。
